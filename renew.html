  7. 효율적인 class명 활용에 대한 고민(예: position : absolute / compatibillity.css)249

    현황 : position값이 들어가는 모든 class에 position이 들어가있는 상태

    수정안 : 공통 속성이 들어가는 class명에 추가해서 쓰는 방식

    .pos.absolute { position: absolute;}
    .pos.relative { position: relative;}
    .pos.fixed { position:fixed}
    .pos.center { left: 50%; top: 50%; transform:translate(-50%,-50%)}

    위와 같이 코드를 작성했다고 가정했을 때 class명을 활용한다면? ➡ class="pos absolute center"

    이 방법을 사용하면 호환성 코드 또한 간결해질 것으로 판단됨

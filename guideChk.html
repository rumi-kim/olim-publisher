CSS 가이드 부분 - General class

  • media query 기준 현행가이드는 Wordpress 테마를 기준으로 작성되어있으나, 가이드 작성 시점과 현재 시점을 비교해볼 때(가이드 작성 시점 : 2020년 / 현재 가이드 수정 시점 : 2022년)
    디바이스의 종류가 다양해지면서 자연스럽게 해상도도 다양해졌기에, 세분화해서 관리하는 것으로 한다.
    다만 기초적인 base 사이즈 지정하되, 갤럭시 z fold 시리즈와 같은 논외케이스는 프로젝트에서 필요할시 추가하여 작성하도록 한다.

    base 사이즈

    /* common window device */
    @media screen and (max-width: 1920px) {}

    /* common small laptop (13inch) */
    @media screen and (max-width: 1280px) {}

    /* old laptop or 13inch laptop, iPad Pro series */
    @media screen and (max-width: 1024px) {}

    /* small tablet - ipad mini */
    @media screen and (max-width: 768px){}

    /* large mobile device */
    @media screen and (max-width: 425px){}

    /* small mobile device - iphone mini series and old device */
    @media screen and (max-width: 360px){}

  • reset
    현행 가이드 항목 일부 수정 후 유지
    수정할 항목 : 전체 font-size
  • font 관련
    폰트 두께 - 내용 보강 후 일부 유지
    word-keep - 스타일선언과 관련된 내용으로 페이지를 분류할 필요성이 보임
    ir_text - 유지
    레이아웃 관련 - 논의 필요
    divider - 폐기
    수평정렬 - 작성중인 신규 가이드를 참조
    부모기준 수직 / 수평 중앙 정렬 - 내용 보강 및 수정하여 유지
